## 📁 What's in This Repository

This repo is the **source archive** of the TOBACCO.com FX system. Every module is a self-contained HTML/JavaScript/WebGL file — open in any browser and it runs.

### `/modules` — The Collection

The `modules/` subdirectory contains every AV Effects screensaver module, each paired with its `.jpg` screenshot thumbnail. Filenames match the live gallery URLs exactly, making it easy to correlate source files with their online counterparts.

**What you'll find inside:**

| Category | Examples |
|---|---|
| 🔬 **Fractal & Chaos Math** | Mandelbrot, Julia, Lorenz attractors, Feigenbaum cascades, bifurcation tapestries |
| 🔷 **Hyperdimensional Geometry** | E8 lattices, Coxeter polytopes, 4D projections, Hopf fibrations, Quaternion engines |
| 🌀 **Kaleidoscopic Systems** | Quasicrystal reactors, Penrose tilings, Voronoi networks, Truchet tile cosmos |
| 🕹️ **Retro Arcade / Atari** | Atari 2600 TIA color palette visualizers, TRON-style vector engines, Pac-Man variants |
| 🧠 **Cybernetic & Neuro** | Neuro-circuitry arcade evolutions, morphogenesis engines, CRISPR vector fields |
| 🌌 **Cosmic Simulations** | Black hole accretion, particle cascades, plasma cathedrals, multiversal spectral engines |
| 🎭 **Color Science** | Chromatic Nexus series, TIA chip palette animations, perceptual illusion laboratories |
| ⚙️ **Mechanical & Maze** | Gear metropolises, hyperbolic labyrinths, puzzle physics reactors, mandala engines |
| 🌊 **WebGL2 Shaders** | Raymarched fractals, spectral manifold engines, Möbius visualizers, plasma fields |

Each module is:
- **Standalone** — a single `.html` file, zero dependencies
- **Browser-native** — runs on any modern browser supporting WebGL/WebGL2
- **Generative** — procedurally animated, never the same twice
- **Archivable** — open-source and self-hostable
