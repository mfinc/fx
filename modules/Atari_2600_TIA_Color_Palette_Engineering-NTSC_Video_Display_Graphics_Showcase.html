Atari 2600 NTSC Color Science and Television Interface Adapter Chip Simulation

This interactive web application presents a breathtaking real-time animation of the Atari 2600 video game console color system. The Atari 2600, released in 1977, used the Television Interface Adapter (TIA) chip to generate video signals and produce its distinctive color palette of 128 NTSC colors organized into 16 hue groups each with 8 luminance levels.

Atari 2600 TIA Color Registers: COLUBK COLUPF COLUP0 COLUP1

The TIA chip color system uses registers: COLUBK for background, COLUPF for playfield, COLUP0 for Player 0, COLUP1 for Player 1. The NTSC color subcarrier operates at 3.579545 MHz.

Six Animation Scenes

  • Scene 0: Scanline Matrix — full palette grid with CRT scanlines and horizontal sync
  • Scene 1: Color Wheel — all 128 colors by hue and luminance in rotating TIA wheel
  • Scene 2: Circuit Board — PCB schematic with TIA chip, CPU, signal traces, data packets
  • Scene 3: Sprite Parade — Atari-style sprites with phosphor glow, trails, missile objects
  • Scene 4: Plasma Wave — NTSC color synthesis using all 128 palette colors
  • Scene 5: Register Cascade — scrolling view of all 128 color registers with NTSC waveforms
